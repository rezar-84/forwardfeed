This alt text generation module is part of the AI module ecosystem and included in DXPR CMS.

Features

The AI Image Alt Text module allows you to generate alt texts for image fields using AI vision models. As it is widget-based, it incorporates a human-in-the-loop process, enabling manual generation and verification of the alt text before saving.

SEO benefits

Generating accurate and descriptive alt texts enhances your website's accessibility and improves search engine optimization (SEO). Alt texts help search engines understand image content, which can improve your site's ranking in image search results. By using this module, you ensure high-quality, consistent alt text generation that adheres to best practices for both accessibility and SEO.

Post-installation

  1. Set up permissions to determine which roles can generate alt texts using AI.
  2. Access the settings page at /admin/config/ai/ai_image_alt_text to customize the prompt or adjust how the image is preprocessed before being sent to the AI.
  3. A button will then appear on image fields for generating alt texts.

Additional requirements

This module uses the AI module as its foundation. It supports any vision model available through providers like OpenAI, Anthropic, and Fireworks AI.
